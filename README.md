# BetterPrompt

A web tool to write better AI prompts.

## Usage

Open `index.html` in a browser.

**Transform Mode:** Answer questions to build a prompt from a rough idea.

**Analyze Mode:** Get feedback on an existing prompt.

## Setup

To use the transform feature, add a Groq API key (free at [console.groq.com](https://console.groq.com)) to line ~499 in `index.html`.

The analyze feature works without an API key.
